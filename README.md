# Mood-Sensing-App

### Server

Run below commands to start the server from root folder
* cd server
* npm install
* npm run start
> Server starts running in http://localhost:4000 by default

### Client
* cd client
* npm install
* npm run start
> Client starts running in http://localhost:3000 by default
